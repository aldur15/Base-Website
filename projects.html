Emotion-Aware Conversational Agent (VR)

Modular VR software enabling real-time interaction with a German-speaking MetaHuman in Unreal Engine. Integrates STT (Whisper), emotion recognition (RoBERTa, Wav2Vec2.0), DiscoLM LLM, and TTS (Tacotron2 via Coqui TTS). Facial animation is synced using Audio2Face. Entire system is Dockerized and deployed on distributed Linux servers.
